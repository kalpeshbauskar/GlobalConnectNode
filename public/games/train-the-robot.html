🤖Train the Robot
Brain Power
Every picture you label becomes training data — an example the robot uses to learn a pattern. The more correctly labeled examples you give it, the better it gets at recognizing brand‑new pictures it has never seen before (that's the test data). But if you label things incorrectly on purpose, the robot learns the wrong pattern too — real AI systems work the same way: garbage in, garbage out.
