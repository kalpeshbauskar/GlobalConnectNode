What's going on here?

Black-and-white pictures on a computer are just grids of tiny squares called pixels. Each pixel is either "off" (light) or "on" (dark). Computers only understand two symbols, 0 and 1, so a whole picture can be written as one long string of binary digits — one 0 or 1 per pixel, row by row. This is called a bitmap.

What's going on here?

Color pictures work the same way, but each pixel needs three numbers instead of one bit. Every pixel is made of three tiny lights: Red, Green, and Blue. Each light can shine at a brightness from 0 (off) to 255 (full power). Mixing the three brightnesses makes every color your screen can show — that's an RGB code, like rgb(255, 0, 0) for pure red.
